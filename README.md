# ts-axios

使用ts造axios的轮子

项目初始化基于`typescript-library-starter`
https://github.com/alexjoverm/typescript-library-starter

## 笔记

### url参数的处理

需要处理以下情况：

- 参数为数组
- 参数为对象
- 参数为Date类型
- 特殊字符支持
- 空值忽略
- 丢弃url上的哈希标记
- 保留url上已存在的参数
